# Linguify — AI Language Translation Tool

A bold, colorful, modern SaaS-style AI translator built with vanilla HTML, CSS, and JavaScript.

## Features
- Real-time translation with a free API fallback chain (no API key required)
- Text-to-speech playback
- Speak-to-type voice input
- Translation history (persisted)
- Download translations as .txt or .pdf (with full Unicode/RTL script support)
- Light/dark mode

Download `index.html` and open it directly in a browser. No build step or server required.
